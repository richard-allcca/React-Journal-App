# Journal App

Una aplicación que simula una especie de diario digital, permite crear todo tipo de anotaciones y agregar imagenes. 

## Temas tocados en este proyecto

   - Maquetación con MaterialUi
   - Redux
   - Store
   - Middlewares
   - Dispatch
   - Actions
   - State
   - Acciones asíncronas
   - RTK Query
   - Redux Toolkit
   - Slices
